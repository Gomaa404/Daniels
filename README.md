# Daniels — Creative Portfolio

A premium, modern portfolio website built for **Jerry Daniels**, a professional UI/UX Designer and Web Developer. This project features a sleek, minimal aesthetic with glassmorphism elements, smooth transitions, and a fully responsive layout.

![Portfolio Preview](img/hero.jpg)

## ✨ Features

- **Premium Design**: Modern aesthetic using Inter, Outfit, and Playfair Display typography.
- **Glassmorphism UI**: High-end visual effects with blurred backgrounds and subtle borders.
- **Dynamic Typing**: Interactive Hero section with a custom typing animation.
- **Interactive Portfolio**: Filterable project gallery for showcasing diverse work.
- **Responsive Layout**: Optimized for all devices, from mobile phones to large desktops.
- **Smooth Navigation**: Scroll-spy enabled navbar with glassmorphism transitions on scroll.
- **Service Showcase**: Dedicated section for services like Web Design, Branding, and Development.
- **Testimonials**: Clean, interactive carousel for client feedback.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flexbox, Grid, and Animations.
- **JavaScript (ES6+)**: Custom typing effects and interactive components.
- **Bootstrap 5**: Responsive grid system and utility classes.
- **FontAwesome**: High-quality iconography.
- **Google Fonts**: Inter, Outfit, and Playfair Display.

## 🚀 Getting Started

### Prerequisites

To view this project locally, you only need a modern web browser.

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/daniels-portfolio.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd daniels-portfolio
   ```

3. **Open in Browser**:
   Simply open the `index.html` file in your preferred browser.
